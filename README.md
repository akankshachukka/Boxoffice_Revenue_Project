# Boxoffice_Revenue_Project
# Project Overview
The objective of this project is to develop a machine learning model using Linear Regression to accurately predict the box office revenue of movies based on various available features.

# Description of dataset
The model will be trained on a dataset containing historical movie data and will aim to identify key factors that impact revenue.
